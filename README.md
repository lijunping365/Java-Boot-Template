# Java-Boot-Template

Spring-Boot 模板项目，助力项目快速启动，包含以下功能：

- spring-boot: 2.3.6
- lombok
- mapstruct
- mybatis-plus
- redis
- 极致的分页查询
- ip 获取，ip2region
- 对象存储，腾讯 cos
- 登录、注册、验证码、用户
